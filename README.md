# Salesforce Account Manager (LWC)

This repository contains a custom Lightning Web Component (LWC) designed for efficient Account management within Salesforce. It provides a clean, user-friendly interface for handling core record operations.

## 🚀 Features
* **Read:** Displays Account records in a structured table.
* **Search:** Real-time search functionality to filter records by name.
* **Update:** Custom Edit Modal using SLDS (Salesforce Lightning Design System) for a native look and feel.
* **Delete:** Integrated record removal logic.
* **User Feedback:** Dynamic loading spinners to improve the user experience during data processing.

## 🛠️ Technical Implementation
* **Framework:** Lightning Web Components (LWC)
* **Design:** Salesforce Lightning Design System (SLDS)
* **Architecture:** Follows Salesforce DX folder structure for professional deployment.
* **Logic:** Uses JavaScript (ES6+) for state management and event handling.

## 📁 Folder Structure
This project uses the standard Salesforce DX directory structure:
`force-app/main/default/lwc/accountManager/`
